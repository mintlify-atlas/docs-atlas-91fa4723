The Stock Request Kanban module enables a lean kanban card system for products that need frequent reordering but cannot use traditional reordering rules. It's ideal for consumables and products with putaway rules.

## Overview

On most companies there are products that must be purchased often but cannot be stored as a usual product because no consumption moves are made. This module allows using stock requests as reordering rules for these products, following the concept of lean kanban cards.

## Key Features

- **Kanban Card System**: Create physical or digital kanban cards for products
- **Barcode Integration**: Scan kanban cards to create stock requests
- **Batch Processing**: Collect multiple cards and process them together
- **QR Code Support**: Generate printable kanban cards with QR codes
- **Stock Request Orders**: Organize kanbans into orders
- **Inventory Kanban**: Manage kanban card inventory

## Configuration

### Barcode Format Settings

The module supports various barcode format configurations via system parameters.

#### Disable CRC Check Digit

If your barcode does not contain a CRC check digit:

1. Enable **Development mode**
2. Go to **Settings > Technical > Parameters > System Parameters**
3. Create a parameter:
   - **Key**: `stock_request_kanban.crc`
   - **Value**: `0`

#### Ignore CRC Check Digit

If you want to ignore an existing CRC check digit:

1. Enable **Development mode**
2. Go to **Settings > Technical > Parameters > System Parameters**
3. Create a parameter:
   - **Key**: `stock_request_kanban.ignore_crc`
   - **Value**: `1`

#### Custom Barcode Format

To change the QR code format:

1. Enable **Development mode**
2. Go to **Settings > Technical > Parameters > System Parameters**
3. Create a parameter:
   - **Key**: `stock_request_kanban.barcode_format`
   - **Value**: Your custom format string

## Usage

### Creating Kanban Cards

1. Navigate to **Stock Requests > Stock Requests Kanban**
2. Click **Create**
3. Fill in the required information:
   - **Product**: Select the product
   - **Quantity**: Enter the standard reorder quantity
   - **Location**: Choose the destination location
4. Click **Save**
5. Click **Print** to generate the physical kanban card
6. Place the printed card in the product storage location

<Info>
The kanban card should be visible and accessible at the product storage location for easy scanning when stock is consumed.
</Info>

### Request Kanban (Individual Cards)

Use this method when creating requests immediately upon consumption:

1. When the product is consumed, take the kanban card
2. Go to **Stock Requests > Order Kanban Card**
3. Scan the card using a barcode scanner
4. The stock request is automatically created

<Note>
This is ideal for just-in-time replenishment workflows where immediate action is needed.
</Note>

### Request Kanban Batch

Use this method for collecting and processing multiple cards together:

1. When products are consumed, collect the kanban cards in a designated location
2. Go to **Stock Requests > Stock Request Orders**
3. Create a new Stock Request Order
4. Click the **Scan** button
5. Scan all collected kanban cards
6. Review and confirm the order

<Info>
Batch processing is efficient for scheduled replenishment runs (e.g., daily or weekly).
</Info>

### Workflow Comparison

| Method | When to Use | Benefits |
|--------|-------------|----------|
| **Individual Request** | Immediate replenishment needed | Quick, real-time processing |
| **Batch Request** | Scheduled replenishment | Efficient for multiple items, better planning |

## Best Practices

### Card Placement

- Place cards in visible, easily accessible locations
- Attach cards to product bins or storage containers
- Use weather-resistant card holders if needed

### Process Design

- Define clear collection points for used kanban cards
- Establish regular scanning schedules for batch processing
- Train staff on proper card handling and scanning procedures

### Quantity Optimization

- Set kanban quantities based on lead times and consumption rates
- Review and adjust quantities periodically
- Consider creating multiple cards for high-consumption items
