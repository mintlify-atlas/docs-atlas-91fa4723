The Stock Request module allows users to request products that are frequently stocked by the company to be transferred to their chosen location. This core module provides the foundation for the entire Stock Request framework.

## Overview

This module enables internal stock requests that integrate with Odoo's procurement system. When a stock request is confirmed, it is automatically evaluated using the procurement rules for the selected location, potentially triggering transfers, purchase orders, or manufacturing orders.

## Key Features

- **Internal Stock Requests**: Request stock transfers for products to specific locations
- **Procurement Integration**: Automatic evaluation using procurement rules upon confirmation
- **Stock Request Orders**: Group multiple requests together
- **Transfer Tracking**: Access related transfers directly from stock requests
- **Access Control**: User and Manager roles with different permission levels
- **Automatic Cancellation**: Cancel pending stock moves when requests are cancelled

## Installation

1. Install the module from Apps:
   ```bash
   pip install odoo-addon-stock-request
   ```

2. Update the app list in Odoo

3. Search for "Stock Request" and click Install

## Configuration

### Access Rights

Users must be assigned to one of the following security groups:

#### Stock Request / User

- Can see their own Stock Requests
- Can see Stock Requests they're granted permission to follow
- Can create and update only their own Stock Requests

#### Stock Request / Manager

- Full management access to all Stock Requests
- Can configure settings
- Can manage all users' requests

### Settings

To configure the module:

1. Go to **Stock Requests > Settings**
2. Configure module parameters as needed

## Usage

### Creating a Stock Request

1. Navigate to **Stock Requests > Stock Requests**
2. Click **Create**
3. Fill in the required fields:
   - **Product**: Select the product to request
   - **Quantity**: Enter the requested quantity
   - **Location**: Choose the destination location
4. Click **Confirm**

<Info>
Upon confirmation, the request is evaluated using procurement rules for the selected location.
</Info>

### Viewing Transfers

When transfers are created from a stock request:

1. Open the Stock Request
2. Click the **Transfers** button
3. View all related stock moves and pickings

### Cancelling a Request

When you cancel a Stock Request:

- All related pending stock moves are automatically cancelled
- Completed transfers remain unchanged

<Warning>
Cancelling a stock request cannot be undone. Ensure this is the intended action.
</Warning>
