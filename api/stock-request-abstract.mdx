## Overview

The `stock.request.abstract` model is an abstract base model providing common fields and functionality for stock requests.

- **Model name**: `stock.request.abstract`
- **Type**: Abstract Model
- **Description**: Stock Request Template
- **Inherits from**: `mail.thread`, `mail.activity.mixin`

## Purpose

This abstract model defines the core structure and shared functionality for stock request models. It provides:

- Common fields for product, quantity, location, and warehouse management
- Route computation logic
- Company and unit of measure validation
- Base onchange methods for field synchronization
- Location parent hierarchy traversal

## Methods

### _compute_product_qty()

Computes the product quantity in the default UoM of the product.

**Dependencies:** `product_id`, `product_uom_id`, `product_uom_qty`, `product_id.product_tmpl_id.uom_id`

**Returns:** None (updates `product_qty` field)

```python
@api.depends(
    "product_id",
    "product_uom_id",
    "product_uom_qty",
    "product_id.product_tmpl_id.uom_id",
)
def _compute_product_qty(self):
    for rec in self:
        rec.product_qty = rec.product_uom_id._compute_quantity(
            rec.product_uom_qty,
            rec.product_id.product_tmpl_id.uom_id,
            rounding_method="HALF-UP",
        )
```

**Example:**

```python
request = env['stock.request'].browse(request_id)
request.product_uom_qty = 5.0  # 5 boxes
request.product_uom_id  # Box (1 box = 12 units)
request.product_qty  # 60.0 units (computed)
```

### _compute_route_ids()

Computes available routes based on product, warehouse, and location configuration.

**Dependencies:** `product_id`, `warehouse_id`, `location_id`

**Returns:** None (updates `route_ids` field)

```python
@api.depends("product_id", "warehouse_id", "location_id")
def _compute_route_ids(self):
    route_obj = self.env["stock.route"]
    routes = route_obj.search(
        [("warehouse_ids", "in", self.mapped("warehouse_id").ids)]
    )
    routes_by_warehouse = {}
    for route in routes:
        for warehouse in route.warehouse_ids:
            routes_by_warehouse.setdefault(warehouse.id, self.env["stock.route"])
            routes_by_warehouse[warehouse.id] |= route
    for record in self:
        routes = route_obj
        if record.product_id:
            routes += record.product_id.mapped(
                "route_ids"
            ) | record.product_id.mapped("categ_id").mapped("total_route_ids")
        if record.warehouse_id and routes_by_warehouse.get(record.warehouse_id.id):
            routes |= routes_by_warehouse[record.warehouse_id.id]
        parents = record.get_parents().ids
        # ruff: noqa: B023
        record.route_ids = routes.filtered(
            lambda r: any(p.location_dest_id.id in parents for p in r.rule_ids)
        )
```

**Route sources:**
1. Product-specific routes (`product_id.route_ids`)
2. Product category routes (`product_id.categ_id.total_route_ids`)
3. Warehouse routes
4. Filtered by rules that deliver to the location or its parents

### get_parents()

Returns all parent locations up to the root location.

**Returns:** `stock.location` recordset - Location hierarchy from `location_id` to root

```python
def get_parents(self):
    location = self.location_id
    result = location
    while location.location_id:
        location = location.location_id
        result |= location
    return result
```

**Example:**

```python
request = env['stock.request'].browse(request_id)
parents = request.get_parents()
for loc in parents:
    print(f"{loc.name} (ID: {loc.id})")
# Output:
# WH/Stock/Shelf 1 (ID: 15)
# WH/Stock (ID: 8)
# WH (ID: 5)
# Physical Locations (ID: 1)
```

### default_get()

Provides default values for warehouse and location.

**Parameters:**
- `fields` (list): List of field names to get defaults for

**Returns:** `dict` - Default values

```python
@api.model
def default_get(self, fields):
    res = super().default_get(fields)
    warehouse = None
    if "warehouse_id" not in res and res.get("company_id"):
        warehouse = self.env["stock.warehouse"].search(
            [("company_id", "=", res["company_id"])], limit=1
        )
    if warehouse:
        res["warehouse_id"] = warehouse.id
        res["location_id"] = warehouse.lot_stock_id.id
    return res
```

**Behavior:**
- If no warehouse is set but company is, finds first warehouse for that company
- Sets location to warehouse's stock location (`lot_stock_id`)

## Onchange Methods

### onchange_warehouse_id()

Updates location and company when warehouse changes.

```python
@api.onchange("warehouse_id")
def onchange_warehouse_id(self):
    if self._name == "stock.request" and self.order_id:
        # Skip if request belongs to an order
        return
    if self.warehouse_id:
        loc_wh = self.location_id.warehouse_id
        if self.warehouse_id != loc_wh:
            self.location_id = self.warehouse_id.lot_stock_id.id
        if self.warehouse_id.company_id != self.company_id:
            self.company_id = self.warehouse_id.company_id
```

**Behavior:**
1. Skips if record is a stock.request with an order (order controls location)
2. Updates location to warehouse's stock location if different
3. Updates company to match warehouse company

---

### onchange_location_id()

Updates warehouse when location changes.

```python
@api.onchange("location_id")
def onchange_location_id(self):
    if self.location_id:
        loc_wh = self.location_id.warehouse_id
        if loc_wh and self.warehouse_id != loc_wh:
            self.warehouse_id = loc_wh
            self.with_context(no_change_childs=True).onchange_warehouse_id()
```

**Behavior:**
- Updates warehouse to location's warehouse if different
- Calls `onchange_warehouse_id()` with special context to prevent cascading changes

---

### onchange_company_id()

Updates warehouse when company changes.

```python
@api.onchange("company_id")
def onchange_company_id(self):
    if self.company_id and (
        not self.warehouse_id or self.warehouse_id.company_id != self.company_id
    ):
        self.warehouse_id = self.env["stock.warehouse"].search(
            [
                "|",
                ("company_id", "=", False),
                ("company_id", "=", self.company_id.id),
            ],
            limit=1,
        )
        self.onchange_warehouse_id()
```

**Behavior:**
- Finds first warehouse for the new company (or without company)
- Triggers warehouse onchange to update location

---

### onchange_product_id()

Updates product UoM when product changes.

```python
@api.onchange("product_id")
def onchange_product_id(self):
    if self.product_id:
        self.product_uom_id = self.product_id.uom_id
```

**Behavior:**
- Sets `product_uom_id` to product's default UoM

## Constraints

### SQL Constraints

- `name_uniq`: Name must be unique per company

### Python Constraints

<Expandable title="_check_company_constrains">
  Validates that all related records (product, location, warehouse, route) belong to the same company or have no company restriction.
  
  **Fields checked:**
  - `product_id.company_id`
  - `location_id.company_id`
  - `warehouse_id.company_id`
  - `route_id.company_id`
  
  **Raises:** `ValidationError` with specific message for each field
</Expandable>

<Expandable title="_check_product_uom">
  Validates that the UoM has the same category as the product's default UoM.
  
  **Checks:** `product_id.uom_id.category_id == product_uom_id.category_id`
  
  **Raises:** `ValidationError` - "You have to select a product unit of measure in the same category than the default unit of measure of the product"
</Expandable>

## Extension Example

To extend stock request functionality with custom fields:

```python
class StockRequestCustom(models.Model):
    _inherit = "stock.request.abstract"
    
    # Add custom field
    priority = fields.Selection([
        ('low', 'Low'),
        ('normal', 'Normal'),
        ('high', 'High'),
    ], default='normal')
    
    # Override method
    @api.onchange('product_id')
    def onchange_product_id(self):
        super().onchange_product_id()
        # Add custom logic
        if self.product_id and self.product_id.categ_id.name == 'Urgent':
            self.priority = 'high'
```
